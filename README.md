# Tapdzi Website

Oficjalna strona `tapdzi.github.io`.

## Lokalnie

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (GitHub Pages)

Repo używa GitHub Actions (`.github/workflows/deploy-pages.yml`).
Po pushu na branch `main` strona publikuje się automatycznie.

## Uwaga

Kod tej strony jest publiczny jako frontend, ale projekt i marka Tapdzi są własnością autora.
